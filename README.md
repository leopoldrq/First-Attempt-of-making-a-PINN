This is my first attempt to work with PINN. Most of the code was generated using Gemini and Grok.
In this code the first Stekloff eigenfunction in the unit disk is predicted by means of a PINN.
You can play with the hyperparameters. 
The code can be run in a Jupyter notebook. I personally run this version using google Colab.

What you see here is just one attempt to start learning and working with a PINN. A lot of the coding was created by using Grok and Gemini. I did minor changes to some of the codes that Grok provided as examples for PINNs. I modify some of them to work in the particular case of the Stekloff boundary value problem on a disk. What you see here predicts the first eigenfunction of such boundary value problem. I am still working on making it work for higher eigenvalues.
